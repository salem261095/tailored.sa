## Overall Layout Philosophy

You want a **Framer-like flow**, which means:

* Fewer rigid sections
* More **fluid storytelling**
* Clear visual rhythm (tight → wide → tight)
* Each section feels intentional, not templated

Think: **“scrolling narrative”**, not “stack of blocks”.

---

## Homepage Structure (Refined)

### 1) Hero (Full-screen, dominant)

**Layout:**

* Centered or slightly left-aligned
* Large headline (2–3 lines max)
* Short supporting line
* 2 CTAs (primary + secondary)

**Optional elements:**

* Small trust line under CTAs (e.g. client names)
* Subtle floating visuals (very minimal)

**Goal:**
Immediate clarity + premium feel

---

### 2) Client Trust Strip (Thin section)

**Layout:**

* Horizontal strip
* Logos in grayscale
* Either static grid OR slow scrolling marquee

**Goal:**
Fast credibility without noise

---

### 3) Services (Editorial layout, not cards)

**Layout:**

* Two columns:

  * Left: sticky title + short intro
  * Right: vertical list of services

Each service:

* Title
* One-line description
* Optional hover → reveal more detail

**Flow:**
User scrolls → services appear progressively

**Goal:**
Feels curated, not “boxed”

---

### 4) Featured Work / Case Studies (Visual anchor)

**Layout:**

* Large image/video per project
* Minimal text:

  * Project name
  * What was done
  * Category tags

Arrangement:

* Alternating layout (left image / right text → flip)

**Goal:**
Breaks text flow + adds visual weight

---

### 5) Process (Structured but light)

**Layout options:**

* Horizontal steps (desktop)
* Vertical timeline (mobile)

Each step:

* Number
* Title
* One sentence

Keep it minimal:
No long paragraphs

**Goal:**
Show method without overwhelming

---

### 6) Brand Story (Editorial block)

**Layout:**

* Wide text block (centered or slightly offset)
* Large typography
* No cards, no icons

Optional:

* Small supporting visual or divider

**Goal:**
Slow the pace, create emotional depth

---

### 7) Outcomes / Impact (Compact grid)

**Layout:**

* 2–4 columns
* Each item:

  * Short title
  * One-line outcome

Avoid:

* Long explanations
* Heavy icons

**Goal:**
Translate services into results quickly

---

### 8) Blog Preview (Clean + minimal)

**Layout:**

* 1 featured post (large)
* 2–3 secondary posts (smaller)

Each:

* Title
* Category
* Date (optional)

No heavy cards → keep it light

**Goal:**
Content credibility without clutter

---

### 9) Final CTA (Strong closing block)

**Layout:**

* Full-width section
* Contrasting background
* Centered content

Content:

* Short headline
* One sentence
* Primary CTA button

Optional:

* Contact shortcuts (WhatsApp, email)

**Goal:**
Clear conversion moment

---

### 10) Footer (Minimal, structured)

**Layout:**

* 3–4 columns max:

  * Brand summary
  * Navigation
  * Contact
  * Social

Keep it:

* Clean
* Low visual weight

## Page-Level Structure (Beyond Homepage)

### Services Page

**Structure:**

1. Intro (headline + short explanation)
2. Services list (same style as homepage but expanded)
3. Optional: process
4. CTA

---

### Work / Portfolio Page

**Structure:**

* Grid or list of projects
* Filter or categories (optional)
* Each item → visual first

---

### About Page

**Structure:**

1. Intro statement
2. Story (long-form, editorial)
3. Philosophy / approach
4. CTA

---

### Blog Page

**Structure:**

* Clean list/grid
* Category filters (optional)
* No heavy sidebar

---

### Contact Page

**Structure:**

1. Intro line
2. Contact options (quick access)
3. Form
4. Location (optional)

---

## Key Structural Principles

1. **Reduce repetition**

   * Avoid multiple sections that say the same thing differently

2. **Alternate density**

   * Dense section → airy section → visual section

3. **Limit section count**

   * ~8–10 sections max on homepage

4. **Avoid “card overload”**

   * Cards only where necessary (blog, maybe work)

5. **Strong hierarchy**

   * Each section must have:

     * clear title
     * one purpose
